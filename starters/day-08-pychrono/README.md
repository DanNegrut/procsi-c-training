`brick_friction.py` is complete and runnable. Read it top to bottom (every block has a
comment), run it, then start tweaking `MU`. See the docstring for the two experiments
to try.

## The two wow demos

Both are official PyChrono demos from the Project Chrono repository. Download the raw
file from GitHub (or ask Claude to fetch it for you) and run it in the `camp` environment.

1. Slider-crank with a plot, Friday's CAD mechanism now actually moving:
   <https://github.com/projectchrono/chrono/blob/main/src/demos/python/mbs/demo_MBS_crank_plot.py>
2. Viper rover driving on rigid terrain:
   <https://github.com/projectchrono/chrono/blob/main/src/demos/python/robot/demo_ROBOT_Viper_Rigid.py>

Tweak ideas: change the motor speed of the crank; change the rover's steering; change
gravity and see what Moon driving looks like.
